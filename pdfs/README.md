📂 Carpeta de Formatos PDF
Sube tus archivos PDF aquí directamente desde GitHub.
Cómo agregar un PDF

Entra a esta carpeta en GitHub
Haz clic en "Add file" → "Upload files"
Arrastra tu PDF y haz clic en "Commit changes"
Listo — aparecerá automáticamente en la página de Formatos

Recomendaciones para los nombres de archivo
Usa guiones en lugar de espacios para que el nombre se vea bien en la página:
✅ solicitud-credito-personal.pdf
✅ formato-imss-pensionados.pdf
❌ Solicitud Crédito Personal.pdf
