# Portafolio de Productos Inbursa

Sitio web profesional para asesores financieros de Inbursa. Diseñado para compartir por WhatsApp con clientes potenciales.

## 📋 Estructura del proyecto

```
.
├── index.html              # Página principal
├── README.md               # Este archivo
├── img/                    # Carpeta de imágenes
│   └── (aquí van las imágenes)
├── docs/                   # Carpeta de PDFs descargables
│   └── (aquí van los manuales en PDF)
└── css/                    # Estilos adicionales (opcional)
```

## 🚀 Características

- ✅ Completamente responsive (móvil, tablet, desktop)
- ✅ 11 productos Inbursa organizados en 3 categorías
- ✅ Cards colapsables para no abrumar al cliente
- ✅ Botón WhatsApp flotante con mensajes dinámicos
- ✅ Diseño moderno con colores Inbursa
- ✅ Animaciones sutiles y profesionales
- ✅ Totalmente funcional sin dependencias externas
- ✅ SEO básico incluido

## ⚙️ Personalización

### 1. Cambiar número de WhatsApp
Busca y reemplaza `526671886581` en el archivo `index.html` con tu número (incluye código de país: +52 México).

### 2. Agregar imágenes
Coloca imágenes en la carpeta `/img`:
```
img/
├── logo.png
├── hero-bg.jpg
├── producto1.jpg
└── etc...
```

### 3. Agregar PDFs descargables
Coloca los manuales en la carpeta `/docs`:
```
docs/
├── gastos-medicos.pdf
├── seguro-vida.pdf
├── credito-hipotecario.pdf
└── etc...
```

Luego vincula en los botones de descarga.

### 4. Cambiar datos de contacto
Edita el footer con:
- Tu nombre/empresa
- Teléfono
- Email
- Redes sociales

## 🔗 Cómo subir a GitHub

1. **Crear repositorio**
   ```bash
   git init
   git add .
   git commit -m "Initial commit: Inbursa portfolio"
   git branch -M main
   git remote add origin https://github.com/tu-usuario/tu-repositorio.git
   git push -u origin main
   ```

2. **Vincular con Vercel**
   - Ve a [vercel.com](https://vercel.com)
   - Conecta tu cuenta GitHub
   - Selecciona este repositorio
   - Click en Deploy
   - Tu sitio estará en vivo en minutos

## 📊 Cómo compartir por WhatsApp

Puedes compartir:
- El link directo: `https://tu-dominio.vercel.app`
- O crear un link corto con bitly/tinyurl
- Los clientes lo abren en el celular y ven todo optimizado

## 🔐 Seguridad

- Ningún dato personal se envía
- Los formularios solo abren WhatsApp
- Las transacciones se hacen directamente con Inbursa
- Cumple con RGPD y protección de datos

## 📝 Notas importantes

- Este sitio es una herramienta de apoyo de ventas

- Todos los productos están regulados por CNBV
